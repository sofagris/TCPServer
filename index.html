TCPServer

A simple TCP Server written in C# for testing, viewing and interacting with TCP data.

Usage:
TCPServer.exe [args]

Args:
--HOST: #Default 0.0.0.0
--PORT: #Default 23
--GUI #Show Windows GUI #Default False
