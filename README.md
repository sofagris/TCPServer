TCPServer
=========

A simple TCP Server for testing, viewing and interacting with TCP data.

Usage:<br>
TCPServer.exe [args]<br>

Args:<br>
--HOST:<IP / Hostname>    #Default 0.0.0.0 <br>
--PORT:<portnumber>       #Default 23<br>
--GUI #Show Windows GUI   #Default False<br>
